FAQs:

  1. Q:Can I use this for school project?
    A:Definitely! This website is entirely made for the purpose of student research and study. This is a website of reliabl sources, so you can complete your project correctly and efficiently.

  2. Q:What do I do if I have taken the quiz and I have more than one learning style?
    A:If you have more than one learning style, it just means that you have more than one way to learn and you are more open to different types of resources.

  3. Q:Can students use these resources at school?
    A:Certainly. This website is entirely for the user's benefit, and it can be used in school, in the library, at home (basically anywhere you want).

  4. Q:I have a problem that I can't find the resource for, what should I do?
    A: You can email us at the Contact section for resources for a specific subject. We are sorry for the inconvenience.

  5. This link says that it is no longer working, could you replace it?
    A:We would love to replace it for you, please contact us and include the link so we can take it down and other users don't run into the same problem.

  6. Q:How long does it take for you to respond back?
    A:It would take us around one day to get back to you because other users might have questions as well.

  7. Q:About what issues can we contact you?
    A:You can contact us to remove a link, to add an useful link, to give us feedback and suggestions, to point out spelling errors, or to ask us any questions about ourselves.

  8. Q:How long did it take to make this website?
    A:This project took just over a week to complete.

  9. Q:What did you use to make this website?
    A:We used HTML to make the webpages, CSS to style, and Java Script for the quizzes.

  10. Q:Should we site this website in our bibliography if we used sources from this website for a project?
    A:Our website directs students to useful sites, but we do not actually provide any study information, so only site the sources you used, not our website.
